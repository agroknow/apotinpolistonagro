Παρουσιαζοντας ενα «Αγροτικο Επιχειρηματικο Οικοσυστημα»

Μια νεα πραγματικοτητα

Μια ιδιαίτερη ευκαιρία δίνεται στους Έλληνες, εν μέσω κρίσης, να επιστρέψουν ή να επενδύσουν στον πρωτογενή τομέα.
Μια επιστροφή που θα συνοδεύεται από όλα τα κατάλληλα εργαλεία ώστε να είναι επιτυχημένη.


Η αφορμη για την εκδηλωση

Η ενασχόληση μέσω ατομικών ή συλλογικών αγροτικών εγχειρημάτων μιας μεγάλης και εξαιρετικά ποικιλόμορφης ομάδας ανθρώπων,
συσσώρευσε γνώσεις, εμπειρίες και ιδέες. Ταυτόχρονα διαπίστωσε - και συνεχίζει να διαπιστώνει - ότι η αγροτική επιχειρηματικότητα
έχει εξαιρετικές προοπτικές αν αντιμετωπιστεί ολιστικά. Παρουσιάζεται πλέον η ανάγκη ώστε οι ιδέες και οι εμπειρίες αυτές να
μοιραστούν με σκοπό τόσο την πληροφόρηση, όσο και την εμπλοκή νέων ενδιαφερομένων σε υφιστάμενα ή νέα επιχειρηματικά εγχειρήματα.


Η εκδηλωση εχει σχεδιαστει ωστε:


• να μας φέρει σε επαφή με την αγροτική ζωή και τις αγροτικές εμπειρίες
• να αναδείξει συγκεκριμένα projects που ήδη “τρέχουν”
• να παρουσιάσει τρόπους πρόσβασης σε όλα τα απαραίτητα εργαλεία για την επιτυχημένη
ενασχόληση με την αγροτική επιχειρηματικότητα
• να αποτελέσει αγρο – επιχειρηματικό κόμβο μεταξύ αγροτών, επιχειρηματιών και επενδυτών (AgroHub)
• να αναδειχθεί μια ομάδα νέων αγροτών η οποία θα τεθεί υπό την ομπρέλα του «Από την πόλη
στον αγρό» στα πλαίσια μιας αγροτικής θερμοκοιτίδας


Απευθυνεται σε:

Νέους παραγωγούς που θέλουν να κάνουν την επιχειρηματική μετάβαση από την πόλη στην ύπαιθρο.
Υφιστάμενους παραγωγούς που ενδιαφέρονται να συμμετέχουν σε κάποια από τα εγχειρήματα.
Επενδυτές που ενδιαφέρονται να επενδύσουν στον πρωτογενή ή δευτερογενή τομέα.
Επιχειρήσεις του αγροδιατροφικού τομέα.

Ο "πιλοτικός αγρός" μας περιμένει να τον επισκεφθούμε. Φακή, ιπποφαές, ζέα και σιτάρι είναι κάποιες απο τις καλλιέργειες

Στα πλαίσια της εκδήλωσης, μια ομάδα 10 νέων αγροτών θα αναδειχθεί και θα μπει υπό την ομπρέλλα του project "Απο Την Πόλη Στον Αγρό"
